RevoFun — Web Mini Games

Overview

RevoFun is a collection of small web games I developed during the milestone project, which contains the games: Memory Cards, Rock-Paper-Scissors, and Number Guessing. I engineered the core game logic using Vanilla JavaScript and the DOM API, implementing randomized events and real-time state updates. I also designed a fully responsive, mobile-first interface using CSS transitions and Flexbox, ensuring a seamless user experience across all devices without the use of external frameworks.

Method

Development Process

  • Built using HTML, CSS, and vanilla JavaScript without external frameworks to strengthen fundamental frontend skills
  • Implemented DOM manipulation and event-driven programming to handle user input and game logic
  • Applied basic UI feedback and state updates to make gameplay clear and interactive

Features

Memory Cards

  • Card matching logic using JavaScript state tracking
  • Randomized card positions for replayability

Rock-Paper-Scissors

  • Player vs computer gameplay
  • Randomized computer choices

Number Guessing

  • Random number generation and user input handling
  • Feedback on higher/lower guesses
  • Limited attempts for added challenge

Additional features

  • Responsive design for mobile and desktop
  • Basic UI feedback (e.g., win/loss messages, score tracking)
  • Instant game feedback without page reloads

My Contributions

  • Designing the game layout and UI structure
  • Implementing JavaScript game logic
  • Handling DOM interactions and user events
  • Fixing bugs during development and testing
  • Ensuring UI works on different screen sizes

Tech Stack

Vanilla HTML, CSS, and JavaScript — focused on DOM manipulation, event handling, and responsive UI.

RevouShop — E-commerce Platform

Overview

RevouShop is a fully functional e-commerce platform built with a focus on user experience and performance. I developed the entire frontend architecture, utilizing the React Context API for real-time cart state management and Tailwind CSS for a mobile-responsive UI. I also implemented an Admin panel with CRUD capabilities and optimized the app with Server-Side Rendering (SSR). To ensure production quality, I authored automated unit tests with Jest, covering login, cart logic, and admin interactions.

Method

Development Process

  • Developed with React and Next.js to structure the application into reusable UI components.
  • Styled with Tailwind CSS to create a responsive and consistent user interface
  • Implemented state management using React hooks to handle product data and cart interactions
  • Used Next.js routing system to organize pages such as product listings and admin dashboard

Features

Product Browsing

  • Users can browse a list of products displayed with images, prices, and descriptions
  • Refresh the page for a new set of products

Shopping Cart System

  • Users can add products to a shopping cart
  • Cart state is managed with React Context API for real-time updates
  • Users can add and remove products before checkout

Product Management (Admin)

  • Admin dashboard for managing product listings
  • CRUD operations for adding, editing, and removing products
  • Real-time updates to the product catalog

Additional features

  • Responsive design for mobile and desktop
  • Accessible navigation and controls
  • Optimized performance for fast loading times

My Contributions

  • Designing the UI layout and user experience
  • Implementing React components for product cards, pages, and cart functionality
  • Building the shopping cart interaction logic
  • Creating the Admin panel with product management features
  • Writing unit tests with Jest to ensure code quality and reliability
  • Debugging and fixing issues during development

Tech Stack

React, Tailwind CSS, and Firebase — focused on building responsive UIs and real-time data management.
