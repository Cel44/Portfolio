RevoFun — Mini Games Collection

RevoFun is a collection of small browser games created during the milestone project: Memory Cards, Rock-Paper-Scissors, and Number Guessing. The modal summarizes the project, technologies used, key features, and notes on implementation.

Features

  • Memory Cards: Flip matching pairs, score tracking, and timer.
  • Rock-Paper-Scissors: Best-of-three rounds with basic AI opponent.
  • Number Guessing: Random number, hints, and limited attempts.

Tech Stack

Vanilla HTML, CSS, and JavaScript — focused on DOM manipulation, event handling, and responsive UI with Tailwind CSS.

Implementation Notes

  • Modular JS functions per game for easier maintenance.
  • Simple state management using objects and arrays; persisted score in session where appropriate.
  • Accessible controls with keyboard support and clear focus styles.
